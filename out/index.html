Node.js

Node.js is a server-side platform that uses Javascript to power a webserver. It's great for small and large applications, and is also easy because you get to use Javascript on both backend and frontend.

Installation

  • Via installer application
  • On the command line

Recommended Frameworks

  • Express: A dead-simple framework for building webservers
  • Passport: Easy Authentication framework for Node.js and Express
  • Meteor: A cloud-based platform that lets you build webapps using a variant of Node.js

Tutorials

  • Learn Javascript
  • Node Beginner
  • Build a Complete MVC Website with Express
  • Learn Node.js with Interactive Lessons
  • Express.js Tutorial
