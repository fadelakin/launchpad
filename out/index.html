  • Codecademy JS Tutorial

jQuery

jQuery is an insanely popular library that makes it really easy to interact with elements on your HTML page! It takes a little bit of extra time to learn, but once you learn it you'll find that it can be really helpful!

  • jQuery learning center

Node.js

Node.js is a server-side platform that uses Javascript to power a webserver. It's great for small and large applications, and is also easy because you get to use Javascript on both backend and frontend.

Installation

  • Via installer application
  • On the command line

Recommended Frameworks

  • Express: A dead-simple framework for building webservers
  • Passport: Easy Authentication framework for Node.js and Express
  • Meteor: A cloud-based platform that lets you build webapps using a variant of Node.js

Tutorials

  • Learn Javascript
  • Node Beginner
  • Build a Complete MVC Website with Express
  • Learn Node.js with Interactive Lessons
  • Express.js Tutorial

Modules

All node modules are searchable on the NPM database

Python

Python is a widely-used scripting language that is really easy to learn. If you're a beginner, you'll especially love how easy it is to read, and how similar to english it looks.

  • Flask: A microframework that is perfect for the smallest of servers
